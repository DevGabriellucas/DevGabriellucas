## 🧠 What I do

-   🏗️ Data Pipelines & ETL/ELT: Construção de fluxos automatizados para coleta e tratamento de dados.
-   ⚡ Machine Learning Workflows: Desenvolvimento e deploy de modelos preditivos simples.
-   🧩 Data Visualization & Dashboards: Criação de visões estratégicas no Power BI e bibliotecas Python (Seaborn/Plotly).
-   🤖 Data Cleaning & Wrangling: Transformação de dados brutos e "sujos" em informações prontas para análise.
-   📊 Exploratory Data Analysis (EDA): Identificação de padrões, tendências e anomalias em grandes volumes de dados.
-   💡 Business Intelligence: Tradução de problemas de negócio em soluções baseadas em evidências.
-   🔗 Database Architecture: Modelagem de bancos de dados relacionais (SQL) e não-relacionais para performance.
